Project: GiorBaliTour - Rental Mobil Bali
Stack: Next.js 14, TypeScript, Tailwind CSS, Prisma, SQLite

STRUKTUR UTAMA:
- /src/app/[locale]/          : Halaman publik dengan i18n routing
- /src/app/[locale]/(tanian)  : Admin dashboard (protected)
- /src/app/api                : API endpoints (auth, reviews, admin)
- /src/components/ui          : Reusable components (Header, Footer, ReviewForm)
- /src/lib/auth.ts            : Auth utilities (prisma, hash, verify password)
- /src/types/index.ts         : TypeScript interfaces
- /messages                   : Translation files (8 bahasa)
- /prisma                     : Database schema & seeding
